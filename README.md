# Quiz
A fun quiz which gives scores at the end

## Features

* Gives multiple choice questions with a single correct answer
* Keeps a track of the score
* Shows the final score at the end of the quiz
* Shows whether the answer was correct or incorrect for each question.

## Technologies Used

* Python

## How to Run

Clone the repository:

git clone <repository-url>

Navigate to the folder:

cd Quiz

Run:

python main.py
